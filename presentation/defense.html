06 · LANGGRAPH ORCHESTRATION

Конечный автомат вместо линейного пайплайна

classify → requires_graph?

Verified Answer

Conditional edges: низкая уверенность → re-route на retrieval

# LangGraph state machine
graph = StateGraph(AgentState)

graph.add_node("guard_in",  input_guardrails)
graph.add_node("router",   router_agent)
graph.add_node("rbac",     rbac_filter)
graph.add_node("retrieve", hybrid_retriever)
graph.add_node("generate", llm_generate)
graph.add_node("guard_out", output_guardrails)

# Условные рёбра:
graph.add_conditional_edges(
  "router", choose_strategy,
  {"graph": "rbac", "vector": "retrieve"}
)
graph.add_conditional_edges(
  "generate", check_confidence,
  {"low": "router", "ok": "guard_out"}
)
